# My own arduino (serjuino)
Hi, I decided to make my own arduino.  
To do this, I needed the following components:  
  
####  • ATMega328 microcontroller 
####  • Breadboard (i use at 830 points) 
####  • "Dad-Dad" wires
####  • 16 MHz quartz resonator
####  • Capacitor 22 pf
####  • Button 
####  • USB with wires
